# nvaders

a c++ ncurses game

uses make to build

tested on osx and linux [+ requires ncurses](https://www.cyberciti.biz/faq/linux-install-ncurses-library-headers-on-debian-ubuntu-centos-fedora/)


to build and run
```
cd nvaders
mkdir build && cd build
cmake ..
make
./nvaders
```
